Dự án sử dụng Nodejs để code back-end và trả về API

Web có các chức năng như:
 - Đăng nhập, đăng xuất
 - Quản lý thông tin tài khoản
 - Sử dụng JWT, Passport để mã hóa và phân quyền

cách chạy code:
- Bước 1:
```
git clone 
```
- Bước 2:
```
cd web_ban_mo_hinh
```
-  Bước 3:
```
yarn i
```
or
```
npm install
```
-  Bước 4:
```
npm start
```
